TRABAJA CON NOSOTROS

Somos un equipo de Data Engineer y Data Scientist expertos en BigData e Inteligencia Artificial.

Estamos orientados a brindar soluciones de valor agregado a las empresas. Nuestros valores más importantes son la calidad de nuestro equipo y el servicio ofrecido, por eso damos gran importancia a las personas que forman parte del equipo.

OFERTA DE TRABAJO

Serás responsable de:

  • Comunicación con el cliente tanto para entender sus necesidades como para transmitir las propuestas y soluciones desarrolladas por el equipo de ingeniería de datos.
  • Participar en el desarrollo de Data Lake y Data Warehouse, así como en aplicaciones en tiempo real.
  • Automatizar importaciones, exportaciones y transformaciones de datos con orquestadores.
  • Gestión de bases de datos SQL y NoSQL.
  • Construir pipelines y aplicaciones de gestión del dato escalables horizontalmente.

OFERTA DE TRABAJO

Requisitos

  • Persona graduada en ingeniería informática u otros grados con un alto componente tecnológico.
  • Claro entendimiento de arquitectura de software y buenas prácticas de programación como los principios SOLID.
  • Conocimientos avanzados de Scala, Python y SQL que permitan escribir, testear y validar código de forma ágil.
  • Conocimientos en sistemas operativos basados en linux.
  • Experiencia con tecnologías del Ecosistema Hadoop, como HDFS, Spark, Sqoop, entre otras.
  • Capacidad de comunicación tanto con personas técnicas como con perfiles de negocio.
  • Experiencia en orquestación de procesos de exportación y transformación del dato.
  • Conocimientos de sistema de control de versiones.

OFERTA DE TRABAJO

Requirements

  • Devoción al software libre.
  • Experiencia con metodologías agile.
  • Perfil en GitHub demostrando sus habilidades.
  • Conocimientos de herramientas de CI / CD.
